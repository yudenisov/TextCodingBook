Типы данных в JavaScript
Обозначение переменных в JavaScript.
	В языке JavaScript имя переменной является идентификатором. Оно может состоять из прописных и строчных латинских букв A-Za-z и цифр 0-9. При этом первым символом должна обязательно быть буква.
	В JavaScript все имена переменных являются регистрозависимыми. Это значит, что переменные с именами saratov, Saratov и SARATOV – различные.
	На языке JavaScript длина переменных вроде бы не ограничена, однако на практике читать и писать имена переменных, содержащие более 40 символов, затруднительно.
	На JavaScript все переменные имеют только один тип: Variant, который, в отличие от Visual Basic, на подтипы не делится. Все форматные преобразования в JavaScript осуществляются автоматически.
	Для переменных – идентификаторов массивов после его имени идёт выражение в квадратных скобках, определяющее индекс массива. Элементы структуры (объектов) должны адресоваться знаком «.».
	Все переменные в Basic имеют локальную область видимости. То есть, они определены только в тех функциях, в которых используются. Если переменные должны быть объявлены как глобальные переменные, это должно быть сделано явно. Как правило, область действия переменных в JavaScript не выходит за теги <script> … </script> в HTML файле. Это не относится к именам функций, область видимости для которых – весь файл
Объявление переменных.
	Перед тем, как быть использованными, переменные в JavaScript должны быть объявлены, как в языке Си. Объявление переменных осуществляется при помощи ключевого слова Var:
var СПИСОК_ПЕРЕМЕННЫХ
где СПИСОК_ПЕРЕМЕННЫХ — имена переменных либо идентификаторы массивов с индексами, разделённые запятыми.
	Как и в языке С++, переменные при объявлении можно сразу же инициализировать. Пример:
var i = 0;
var nObject = new (MyObject);
var array = [1, 2, 3];
	В первом случае создаётся переменная, и ей присваивается целочисленное значение 0.
	Во втором случае создаётся объект класса MyObject, и его экземпляр присваивается переменной nObject;
	В третьем случае создаётся массив array из 3 чисел;
	Объявление объекта похоже на объявление массива:
var anObject = {
	key1: "value 1",
	key2: "value 2"
};
	Как видим, объявление объекта в JavaScript похоже на объявление хеш-массива в Perl.
