VBSCRIPT:
MsgBox
Функция используется для отображения сообщений из программ на VisualBasic в операционных системах с GUI.
Синтаксис:
	MsgBox( СООБЩЕНИЕ, СТИЛЬ_КНОПОК, ЗАГОЛОВОК )
Описание:
	Функция показывает всплывающее окно, имеющее заголовок ЗАГОЛОВОК и выводящее на панель определённое сообщение. Панель сообщения оформляется в соответствии с определённым стилем, указанным в таблице ниже. Стили могут комбинироваться при помощи операторов "ЛОГИЧЕСКОЕ AND".
	Функция возвращает номер нажатой кнопки.Возможные кнопки: "Ok", "Cancel", "Abort", "Retry", "Ignore", "Yes", "No".
InputBox
Функция используется для получения информации от пользователя, вводимой в диалоговой панели (в строке редактирования). Ввод данных завершается выбором кнопки Ok или нажатием клавиши "Enter".
Синтаксис:
	СТРОКА = InputBox( ПРИГЛАШЕНИЕ, ЗАГОЛОВОК, ПО_УМОЛЧАНИЮ, X, Y )
где
	СТРОКА -- вводимая строка с данными;
	ПРИГЛАШЕНИЕ -- ткекст на панели, который выводится в справочных целях для информирования пользователя о запросе;
	ЗАГОЛОВОК -- заголовок диалогового окна;
	ПО_УМОЛЧАНИЮ -- присваемое переменной СТРОКА значение в случае, когда не введено ни одно значение пользователем (он просто нажал клавишу "Enter");
	X -- горизонтальная координата панели относительно главного окна;
	Y -- вертикальная координата панели относительно окна;
