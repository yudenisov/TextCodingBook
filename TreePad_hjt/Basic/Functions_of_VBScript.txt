Функции VBScript
	В этом разделе рассматриваются различные встроенные функции языка VBScript. Эти функции могут использоваться для создания динамических, активных веб-документов.
Функция Abs
	Эта функция может быть использована для получения абсолютного значения переменной или числа. Например, Abs(-1) возвращает 1.
Функция Array
	Функция Array  может использоваться для быстрого создания массива. Ниже показано, как использовать эту функцию. После выполнения двух следующих строк:
Dim MyArray As Variant
MyArray = Array( “Element1”, “Element2”, “Element3”)
Будет создан массив, содержащий три элемента: [Element1], [Element2], [Element3]
Функция Asc 
	Эта функция возвращает ASCII-код символа или первого символа в строке. Например, Asc(“A”) возвращает 65, а Asc(“Russia”) — 82.
Функция Atn
	Функция Atn возвращает арктангенс аргумента.
Функция CBool 
	Функция CBool возвращает булевское выражение, результат которого зависит от аргумента. Например, CBool(Var1 = Var2) возвращает True, если переменные Var1 и Var2 содержат одно и то же значение, и False в противном случае.
Функция CByte 
	Функция CByte преобразует число, переданное ей в качестве параметра, в байт. Например, число 123.456 будет преобразовано в 123.
Функция CDate 
	Функция CDate преобразует аргумент в вариант с подтипом Date. Перед передачей документа можно использовать функцию IsDate (см. ниже) для определения того, можно ли преобразовывать данное выражение в подтип Date.
Функция CDbl
	Преобразует аргумент в вариант с подтипом Double.
Функция Chr
	Эта функция возвращает ASCII-символ по ASCII-коду. Например, Chr(65) возвращает “A”.
Функция CInt
	Эта функция преобразует аргумент в вариант с подтипом Integer. Например, CInt(1234.567) возвращает 1235.
Функция CLng
	Функция CLng преобразует аргумент в вариант с поддержкой Long. Например, CLng(12345.67) возвращает 12346.
Функция Cos
	Эта функция используется для вычисления косинуса угла, переданного ей в качестве параметра.
Функция CSng
	Функция CSng преобразует переданное ей в качестве аргумента число в вариант с подтипом Single. Например, CSng(12.1234546) возвращает 12.12346.
Функция CStr 
	Функция CStr преобразует переданное ей в качестве аргумента число в вариант с подтипом String. Например, CStr(123.456) возвращает «123.456».
Функция CVErr
	Эта функция используется для того, чтобы вернуть указанный в качестве параметра код ошибки. 
Функция Date
	Возвращает дату
Функция DateSerial
	Используется для различных вычислений, связанных с датой. Например, используя числовые выражения, можно вычислять дату на несколько дней вперёд или назад (просто вычитая или прибавляя необходимое число дней).
Функция DataValue
	Функция DataValue преобразует переданное ей в качестве параметра выражение в вариант с типом Date. Например, DataValue(“March 18, 1997”) возвращает 3/18/1997. Если год не указан, его значение берётся из системных часов.
Функция Day
	Эта функция используется для определения дня указанной в качестве параметра дате. Например, Date(“3/18/1997”) возвращает 18.
Функция Exp
	Эта функция возвращает число e (основание натурального логарифма), возведённое в указанную степень. Например, Exp(1) возвращает 2.71828182845905.
Функция Hex
	Функция Hex возвращает шестнадцатеричное значение аргумента. Например, Hex(10) возвращает A.
Функция Hour
	Эта функция возвращает число часов в данном выражении. Например, Hour(“12:34:56”) возвращает 12.
Функция InputBox
	Функция InputBox используется для получения информации от пользователя, вводимой в диалоговой панели. Эта функция имеет следующий синтаксис:
InputBox(<Prompt>,  <Title>, <Default>, <X>, <Y>)
где: 
* <Prompt> – строка, поясняющая вводимые данные;
* <Title> – заголовок диалоговой панели;
* <Default> – значение по-умолчанию (если пользователь не указал никакого значения)
* <X> – горизонтальная координата панели
* <Y> – вертикальная координата панели
Функция InStr
	Функция InStr используется для получения местоположения одной строки внутри другой. Эта функция имеет следующий синтаксис:
InStr(<BeginPosition>, <String1>, <String2>, <ComparisonType>)
где:
* <BeginPosition> — необязательный аргумент, задающий начальную позицию текста;
* <String1> — строка, в которой производится поиск;
* <String2> — искомая строка
* <ComparisonType> — необязательный документ, задающий способ сравнения. 0 — бинарное сравнение, 1 — сравнение с игнорированием регистров символов. По-умолчанию используется 0.
Функции Int, Fix
	Функции Int и Fix используются для преобразования аргументов в целочисленные выражения. Единственным различием между этими функциями является то, что Int преобразует отрицательные числа с дробной частью к наименьшему целому, а Fix — к наибольшему целому. 	Функция IsArray возвращает True, если её аргумент является вариантом с типом Array, и False в противном случае.
Функция IsDate
	Эта функция возвращает True, если аргумент является допустимой датой, и False в противном случае.
Функция IsEmpty
	Функция IsEmpty возвращает True, если аргумент был инициализирован, и False в противном случае.
Функция IsError
	Функция IsError возвращает True, если аргумент содержит код ошибки, и False в противном случае.
Функция IsNull
	Функция IsNull возвращает True, если аргумент имеет значение NULL, и False в противном случае.
Функция IsNumeric
	Функция IsNumeric возвращает True, если аргумент является цифровым значением, и False в противном случае.
Функция IsObject
	Функция IsObject возвращает True, если аргумент является ссылкой на объект OLE Automation, и False в противном случае.
Функция LCase
	Преобразует все символы в строке к символам нижнего регистра.
Функция Left
	Функция Left возвращает определённое число символов из строки. Например, Left(“VBScript for Internet”, 8) возвращает “VBScript”.
Функция Len
	Функция Len возвращает число символов в строке. Замечание: Пустая строка имеет в VBA количество символов, равное 1.
Функция Log
	Возвращает натуральный логарифм аргумента.
Функции LTrim, RTrim, и Trim
	Эти функции удаляют лишние пробелы из строки. Функция LTrim удаляет предшествующие пробелы, RTrim — последующие, а Trim — и предшествующие, и последующие. 
Функция Mid
	Возвращает определённое число символов строки. Имеет синтаксис:
Mid(<начальная строка>, <позиция>, <длина новой строки>)
Например, Mid(“VBScript for Internet”, 1, 8) возвращает “VBScript”.
Функция Minute
	Возвращает число минут. Например, Minute(“12:34:56”) возвращает 34ю
Функция Month
	Функция Month возвращает месяц. Например, Month(“3/18/1997”) возвращает 3.
Функция MsgBox
	Эта функция используется для отображения сообщений. Синтаксис этой функции следующий:
MsgBox(<MessageBoxPrompt>, <ButtonStyle>, <Title>)
	Параметр MessageBoxPrompt задаёт текст сообщения, вызываемого в панели, параметр Title — заголовок панели, а параметр ButtonStyle описывает различные состояния панели сообщений, описанные в следующей таблице:

	Функция Now возвращает текущую дату и время.
Функция Oct
	Функция Oct возвращает восьмеричное значение аргумента. Например, Oct(10) возвращает 12.
Функция Right
	Функция Right возвращает определённое число символов с правой стороны строки. Например, Right(“VBScript”, 6) возвращает “Script”.
Функция Rnd
	Функция Rnd возвращает случайное число в диапазоне от 0 до 1. Перед вызовом этой функции необходимо вызвать процедуру Randomize.
Функция Second
	Функция Second возвращает число секунд. Например, Second(“12:34:56”) возвращает значение 56.
Функция Sgn
	Возвращает знак числового выражения. Если выражение равно 0, возвращается значение 0, если оно меньше 0, то возвращается -1, в противном случае возвращается 1.
Функция Sin
	Функция Sin возвращает синус угла. Например, Sin(Pi) возвращает 0.
Функция Sqr
	Функция Sqr возвращает квадратный корень аргумента. Аргумент не может быть отрицательным числом. 
Функция Str
	Эта функция преобразует числовое выражение в строку.
Функция StrComp
	Эта функция используется для сравнения двух строк и имеет следующий синтаксис:
StrComp(<String1>, <String2>, <ComparisonMethod>)
	После сравнения функция StrComp возвращает 0, если обе строки идентичны, -1, если <String1> меньше, чем <String2>, и 1 в противном случае. Аргумент <ComparisonMethod> является необязательным и указывает метод сравнения. Если значение этого аргумента равно нулю, выполняется бинарное сравнение (значение по-умолчанию), если значение равно 1, то выполняется сравнение без учёта регистра букв.
Функция String
	Эта функция используется для заполнения строки определённым числом символов. Например, String(10, “A”) вернёт строку “AAAAAAAAAA”.
Функция Tan
	Функция Tan возвращает тангенс угла. Например, Tan(0) возвращает 0.
Функция Time
	Эта функция возвращает текущее время.
Функция TimeSerial
	Эта функция используется для различных вычислений, связанных со временем. В качестве параметров этой функции указывается три числа, соответствующие часу (0-23), числу минут (0-59) и числу секунд (0-59).
Функция TimeValue
	Эта функция используется для преобразования аргумента к типу Date.
Функция UCase 
	Эта функция используется для преобразования строчных букв в прописные буквы. 
Функция Val
	Эта функция может использоваться для преобразования строчного значения в цифровое значение. Например, функция Val с аргументом: «1234 — это число» вернёт число 1234.

	С помощью этой функции можно определить тип переменной. Например, если переменная CurDate имеет тип Date, функция VarType(CurDate) возвратит значение 7. Значения, возвращаемые функцией VarType, для различных типов данных перечислены в следующей таблице:

	Функция Weekday возвращает номер от 1 до 7, которое соответствует дням недели, как это показано в следующей таблице:

Код дняДень недели1Воскресенье2Понедельник3Вторник4Среда5Четверг6Пятница7СубботаФункция Year
	Эта функция возвращает год, извлекая её из строки, содержащей дату. Например, функция Year(“3/18/97”) возвратит “1997”.
